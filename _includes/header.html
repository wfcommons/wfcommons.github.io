  • They Use WfCommons

WfCommons seeks to foster scientific workflow research and development. It provides open-source datasets of scientific workflow instances and tools for exploring these datasets, for generating synthetic workflow instance, and for producing executable workflow benchmarks. The end-goal is to enable the development of new techniques and runtime systems for the efficient execution of scientific workflows on complex distributed infrastructures.
